# 🌸 SafeHer - Women Safety Website

## 📌 Project Description
SafeHer is a full-stack web application designed to enhance women's safety by providing real-time emergency assistance and NGO support. It allows users to send SOS alerts, share their location, and connect with nearby NGOs instantly.

---

## 🚀 Features
- 🚨 SOS Emergency Button (Instant Alert)
- 📍 Live Location Tracking
- 🗺️ Interactive Map with NGO Locations
- 🔔 Real-time Notifications (Socket.IO)
- 📝 Incident Reporting System
- 🔐 Secure Login & Authentication (JWT)
- 🏥 NGO Directory & Support System

---

## 🛠️ Technologies Used

### Frontend:
- HTML

- JavaScript

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB

### Other Tools:

## 🧠 How It Works

1. User logs into the system
2. In emergency, user presses SOS button
3. Location is captured using GPS
4. Alert is sent to server
5. NGOs/Admin receive real-time notification
6. Help can be provided quickly

---

## 🎯 Project Objective

To provide a fast and reliable safety solution for women using modern web technologies and real-time communication.
